# Menu photos

Put menu item images here, then reference them in the admin "Фото (путь или URL)"
field as `images/<filename>` — for example `images/black-coffee.jpg`.

Tips:
- Use lowercase names without spaces (e.g. `cafe-den.jpg`, not `Cafe Đen.JPG`).
- Compress photos to ~100–300 KB and ~800px wide — they load faster and the menu
  stays well within GitHub's free limits.
- Both relative paths (`images/foo.jpg`) and full URLs (`https://…`) work.

This folder is served free by GitHub Pages alongside the site, so no Firebase
Storage / Blaze plan is needed.
